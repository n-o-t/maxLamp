maxLamp

app controled RGB led matrix. node + express + socket.io + johnny-five

what's in ?

  • individual or global color picker that changes the RGB neopixel matrix on the fly
  • pomodoro feature

how does it work ?

a matrix of 5x8 individually addressable rgb leds are controlled by a web app.

  • an 8x5 led matrix is plugged on a arduino Uno flashed with Firmata firmware.
  • the arduino board is connected to the raspi with a usb cable (data communication only) (important : arduino should have is own power supply, DO NOT rely on the raspi to do that, it works till a certain amount of led brightness)
  • a raspberry pi hosts the app running node.js and express
  • the serial port connection is handled by johnny-five (using npm serialport)
  • real time data communication is handled by socket.io

see Adafruit's page on NeoPixel leds https://learn.adafruit.com/adafruit-neopixel-uberguide/
